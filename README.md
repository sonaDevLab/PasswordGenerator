# 🔐 Password Generator

Aplicación **web simple** que permite generar contraseñas seguras de manera personalizada, con opciones para longitud, tipos de caracteres y un historial de contraseñas generadas.

---

# 📌 Descripción

Password Generator es una herramienta web que ayuda a los usuarios a crear contraseñas fuertes y seguras para sus cuentas. Los usuarios pueden seleccionar la **longitud**, los **tipos de caracteres** (mayúsculas, minúsculas, números y símbolos) y mantener un **historial** de contraseñas generadas para fácil acceso.

La aplicación está desarrollada usando **JavaScript, HTML y CSS**, ofreciendo una experiencia interactiva y sencilla desde el navegador.

---

# ✨ Funcionalidades

🔹 Elegir longitud de la contraseña  
🔹 Incluir o excluir: mayúsculas, minúsculas, números y símbolos  
🔹 Generar contraseñas seguras con un solo clic  
🔹 Copiar contraseña al portapapeles  
🔹 Historial de contraseñas generadas  
🔹 Interfaz intuitiva y responsive

# ⚙️ Cómo usar

1. Abrir `index.html` en un navegador moderno.  
2. Elegir la **longitud** de la contraseña deseada.  
3. Seleccionar los **tipos de caracteres** que deseas incluir.  
4. Hacer clic en **"Generar contraseña"**.  
5. Copiar la contraseña o revisarla en el **historial**.

# ⚠️ Retos enfrentados

- Crear una lógica que permita combinar diferentes tipos de caracteres de forma segura.  
- Evitar duplicados o errores en la generación de contraseñas cuando se cambian los filtros.  
- Mantener un **historial funcional** sin usar base de datos.  
- Lograr un **diseño responsive** que funcione en móviles y escritorio sin frameworks externos.  
- Mejorar la **experiencia de usuario** al copiar contraseñas y visualizar el historial.
